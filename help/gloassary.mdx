## Glossary of Terms

This glossary defines key terms related to web scraping and ScrapeAutomate features.

| **TERM**                 | **DEFINITION**                                                                   |
| ------------------------ | -------------------------------------------------------------------------------- |
| **API**                  | A set of protocols for software applications to communicate with each other.     |
| **Proxy Rotation**       | The use of multiple proxies to avoid IP bans by rotating the IP addresses.       |
| **JavaScript Rendering** | The process of executing JavaScript code on a webpage to render dynamic content. |
| **CAPTCHA**              | A security feature that differentiates between human users and bots.             |
| **Geotargeting**         | Simulating browsing from specific locations to access region-specific content.   |
| **Scraper**              | A tool or script used to automatically extract data from websites.               |
| **Rate Limiting**        | A restriction that limits the number of requests a user can make to a server.    |
| **API Key**              | A unique identifier used for authenticating requests to an API.                  |
